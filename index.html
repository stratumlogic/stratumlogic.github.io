About Us

Stratum Logic is a small company based in Cincinnati, Ohio that focuses on Java and C# development, but also can develop in Javascript, Ruby, Python, Groovy, and more.

Our services include developing desktop, web applications, mobile applications for the iPhone or Android, and webservices.

If this is your first time needing software developed, that's no problem. Stratum Logic can help you choose a platform, language, system setup, security and so on.

Requirements

Whether you want a new app developed, or you need to update an existing app, one of the most important tasks is defining what needs to be done.

Stratum Logic can help you find out what all the questions are, and then help you find out what the answers are.

Interface Design

Applications should be simple to use. We all know that, but often the applications we build for ourselves fall short of our wishes. Often our main focus is to get the core functionality done first, and then focus on usability afterward. And then when the core functionality expands, the usability concerns get pushed back, again and again.

The good news is that Stratum Logic can help. Most times, applications that are easy to use can be made without complex programming. Just smart choices. There are times, though, that smart choices alone can't solve making the application easy to use-- and when that is the case, we can help.
